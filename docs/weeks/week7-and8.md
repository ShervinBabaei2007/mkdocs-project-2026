# User Flow & Low Fidelity Wireframes

This week was quite heavy in terms of workload. It is also worth noting that our instructor Chris had been away sick for 2-3 weeks prior, so we were catching up on a significant amount of work all at once.

Our instructor asked us to create 3 user flows based on the 3 user stories we had previously developed. Personally, this was a manageable task for me since I already had a clear vision of how users would interact with the product.

Following the user flows, we moved on to low fidelity wireframes. The instructor suggested doing them on paper, however I found that approach difficult to work with, so I switched to Figma which significantly sped up my process and helped me better visualize the layout.
